# Explicación Ejercicio 1

En este ejercicio hicimos un servidor básico con Node.js y Express. 

### Pasos:
1. **Servidor**: Usamos `express` para levantar un servidor en el puerto 3001. Configuramos carpetas estáticas para que el navegador pueda leer los estilos y scripts.
2. **Consola**: Al iniciar el servidor, se muestran los mensajes "Hola mundo desde Node.js" y "Fin" en la terminal, como pedía la consigna.
3. **Frontend**: Creamos un HTML con Bootstrap para que sea responsive.
4. **Modo Oscuro**: Implementamos una lógica en la carpeta `Context` para guardar la preferencia del usuario en el `localStorage`.
5. **Modularidad**: Separamos la lógica del tema en un archivo aparte para que el código sea más prolijo.
6. **Scroll**: Agregamos un botón que aparece cuando bajás mucho para volver arriba al toque.
