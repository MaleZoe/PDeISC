# Explicación Ejercicio 3

Este es igual al anterior pero usando funciones flecha para todo.

### Puntos clave:
1. **Funciones**: Definimos `sumar`, `restar`, etc., como constantes que guardan funciones flecha.
2. **Validación Real-Time**: Agregamos un listener de `input` que chequea si estás poniendo cualquier verdura. Si está mal, el borde se pone rojo (clase `is-invalid` de Bootstrap).
3. **Interactividad**: Ahora podés poner tus propios números en la página y ver el resultado sin recargar.
4. **Backend**: El server sigue logueando los resultados hardcodeados del ejercicio original para cumplir la consigna a rajatabla.
