Sobre el Proyecto 4

Objetivos de Aprendizaje

Este proyecto demuestra el poder de la manipulación dinámica del DOM sin usar librerías externas. Los conceptos clave incluyen:

  • createElement: Instanciar nodos en memoria.
  • setAttribute: Modificar el estado del atributo HTML real.
  • Sistema de Auditoría: Rastrear cambios de estado mediante diffs visuales.

Tecnologías Utilizadas

Tecnología Uso en el Proyecto
Node.js & Express Servidor backend y gestión de rutas estáticas.
Vanilla JavaScript Lógica de manipulación del DOM y gestión de estado.
Bootstrap 5 Estructura base y componentes de interfaz.
CSS3 Moderno Animaciones, layouts de Grid y variables (Custom Properties).

¿Cómo funciona createElement?

  1. Instanciación: document.createElement('a') crea un objeto de elemento en la memoria del navegador.
  2. Configuración: Se aplican atributos y contenido (setAttribute, textContent) mientras el nodo aún es "huérfano".
  3. Inserción: appendChild() o prepend() inyectan el nodo en el árbol del DOM activo, haciéndolo visible.
  4. Referencia: JS mantiene una referencia al nodo real, permitiendo mutaciones posteriores en tiempo real.

¿Por qué setAttribute?

Usamos setAttribute en lugar de la asignación directa de propiedades (el.href = ...) para garantizar que los cambios se reflejen exactamente en el atributo del marcado HTML. Esto es vital para la consistencia entre el estado de JavaScript y lo que el motor del navegador renderiza en el inspector de elementos.
